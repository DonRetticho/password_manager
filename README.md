First personal project for Boot.dev

Disclaimer: This is a Proof of Concept for myself! This is not intended to be a professional password manager!
So please do not use it for that and look for something professional if you want to use a password manager!

This is a password manager with a simple customTKinter UI. 

When started the UI asks to set a master password. After that a service/website, username and password can be entered.
Service/website and username are required fields. If they are not entered a status label will tell that they are required. 
If no password is entered it will be generated by random (with securities library). The user can enter how many characters the 
password should have (default is 14). The minimum length is 14 characters and the maximum length is 40.
The service, username and password are saved in a table (tk tree) and are grouped by service. If multiple
usernames and passwords exist for the same service it is possible to enter multiple for one service. 
It is also possible to search for services (with live filtering). If a password is no longer required it can be deleted.
When a cell is selected it is possible to edit it with "F2" (the service cannot be edited).
When double clicking on the username or password it gets copied to the clipboard.

The passwords are saved in a JSON file and the salt for the masterpassword is in a .bin.
